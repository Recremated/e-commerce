# E-Commerce Website

A modern e-commerce website built with React, Vite, and Tailwind CSS.

## Tech Stack

- React 18
- React Router
- Tailwind CSS
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
